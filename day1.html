Day 1 정리

· 5강 : 처음으로 html 파일 생성 후 Hello World를 입력하여 간단하게 파일을 실행해보았다.

· 6강 : 글씨를 강조할 때 사용하는 strong 태그와 밑줄을 표현하고 싶을 때 사용하는 underline 태그에 대하여 공부하였다.

· 7강 : f12 버튼을 눌러 page source을 한 번 보고 h1~h6 태그에 대하여 공부하였다.

· 8강 : 웹페이지들을 구성하는데 어떤 종류의 tag들이 많이 사용되는지와 더불어 페이지 당 평균 사용하는 태그의 수를 확인할 수 있었다.

· 9강 : 한 문단 내에서 줄바꿈을 하는데 사용하는 br 태그와 문단을 나눌 때 사용하는 p 태그에 대하여 공부하였다.
또 br 태그는 end tag가 없으며 두 번 사용하면 문단을 바꾸는 것처럼 보이는 효과를 가져올 수 있지만, 실제로 문단이 생기는 것이 아니기 떄문에 스타일 적용시 문제가 발생할 수 있음을 인지하였다.

· 10강 : 고화질의 이미지를 무료로 사용할 수 있는 사이트와 img 태그를 사용하는 방법을 배워 이미지를 첨부해봤다.
